Units in CSS

Units in CSS are used when expressing measured value such as width, margin font sizes, line height, etc. Some units for example as for width, vdw, it is better to use percentage in a lot of instances. Here are the list of common units used in CSS :

  • em- font size pf parent
  • ex- x-height of element font
  • lh-line height
  • vw - percentage view width
  • vh- viewport height
